# 🧮 Calculator

## 📖 Overview
This project is a **simple Calculator** built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧹 Clear / Reset functionality  

---

## 🧰 Technologies Used
- **HTML5**  
- **CSS3**  
- **JavaScript**
